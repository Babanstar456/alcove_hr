Import Guidelines

File requirements:

  • Accepted formats: .xlsx, .xls, .csv
  • Row 1 must be the header row
  • Max file size: 5 MB
  • All imported tasks start at P1
  • Your Employee ID is saved as creator

Required columns:

  • job_designation
  • project_id
  • location_id
  • position_type — NEW or REPLACEMENT
  • educational_qualification
  • experience_required
  • gender_preference — ANY / MALE / FEMALE
  • monthly_gross_salary
  • number_of_positions

Optional columns: job_responsibilities, age, additional_note, reporting_authority_id
